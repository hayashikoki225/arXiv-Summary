### [1] Data Preservation in High Energy Physics: Global Report 2026
- **著者**: Matthew Bellis et al.
- **リンク**: https://arxiv.org/abs/2607.06775
- **要約**: DPHEP workshop 2026 のグローバル報告。HEP のデータ保存について、legacy data の再解析、FAIR 原則、open data、長期資金、実験終了後のアーカイブ運用を整理。特に AI/ML によるメタデータ抽出・データキュレーション・workflow optimization への移行が強調されている。物理結果ではないが、将来の再解析可能性、特に大型実験の public release 戦略を考える上では重要。
- **キーワード**: data preservation, open data, FAIR, DPHEP, AI/ML

### [2] The Dark Photon in the PHELEX Experiment
- **著者**: A. V. Kopylov, I. V. Orekhov, V. V. Petukhov, A. E. Solomatin
- **リンク**: https://arxiv.org/abs/2607.07074
- **要約**: PHELEX の multi-cathode counter で 2023–2026 年に測定した single-electron count rate の日周変動を、10–40 eV 質量の dark photon の可能な兆候として解釈。Fe と Al cathode の相関から、磁気的性質に依存する dark matter model よりも、電場/longitudinal field 経由で相互作用する kinetic-mixing dark photon 的な解釈を支持すると主張。かなり挑戦的な主張なので独立検証が鍵だが、低エネルギー single-electron 系の異常探索としては面白い。
- **キーワード**: dark photon, PHELEX, single electron, diurnal modulation, low-mass DM

### [3] Search for an isoscalar partner of the $Z_c(3900)$ in $e^+e^-\to\pi^+\pi^-\eta J/\psi$
- **著者**: BESIII Collaboration
- **リンク**: https://arxiv.org/abs/2607.07446
- **要約**: BESIII が 4.18–4.95 GeV の 20.3 fb$^{-1}$ データで $e^+e^-\to\pi^+\pi^-\eta J/\psi$ を 6.0σ で観測し、$Z_c(3900)$ の isoscalar partner 候補 $X(3900)\to\eta J/\psi$ を探索。$X(3900)$ の有意な信号は見えず、質量・幅仮定ごとの Born cross section × branching fraction 上限を設定。exotic hadron spectroscopy としては堅実な null result。
- **キーワード**: BESIII, exotic hadron, $Z_c(3900)$, $X(3900)$, charmoniumlike

### [4] First Demonstration of a Hybrid Cherenkov and Scintillation Detector in a Proof-of-Principle Axion Search at a Beam Dump
- **著者**: Darcy A. Newmark
- **リンク**: https://arxiv.org/abs/2607.07476
- **要約**: CCM の 10 ton 液体 Ar 検出器を用いた Ph.D. thesis。PMT の 80% を wavelength shifter 付き、20% を uncoated とすることで、シンチレーション光と Cherenkov 光を波長・タイミングで分離する hybrid optical detector を実証。$^{22}$Na calibration の詳細モデリング、sub-MeV electron に対する event-by-event Cherenkov/scintillation 分離、ML による位置分解能 ~5 cm・エネルギー分解能 ~10% at 1 MeV を達成。ALP 探索では excess はないが、従来 CCM より小さい exposure でも背景抑制により一部の ALP parameter space を改善。DUNE supernova neutrino への応用も議論しており、低エネルギー LAr 光検出の技術論としてかなり重要。
- **キーワード**: hybrid Cherenkov-scintillation, liquid argon, CCM, ALP, DUNE, supernova neutrino

### [9] Machine learning pipeline for identifying tracks of muons and hadrons at GRAPES-3 muon telescope
- **著者**: Mansi Talwar, Sambit Sarkar, Pravata K. Mohanty
- **リンク**: https://arxiv.org/abs/2607.07263
- **要約**: GRAPES-3 muon telescope で、air shower 中の secondary muon と punch-through hadron を分離する ML pipeline。単一粒子分類では XGBoost が 88.7% accuracy、multi-particle event では detector hit を node とする GNN/edge convolution で hit-level の muon/hadron 分類を行う。宇宙線 muon telescope の背景分離としては素直な発展だが、detector hit graph を使う点はニュートリノ検出器の reconstruction とも相性がよい。
- **キーワード**: GRAPES-3, muon telescope, hadron background, GNN, cosmic ray

### [10] The Deep Learning Cosmic Ray Energy Reconstruction Pipeline for the GRAPES-3 Experiment
- **著者**: Sambit Sarkar, Mansi Talwar, Pravata K. Mohanty
- **リンク**: https://arxiv.org/abs/2607.07265
- **要約**: GRAPES-3 の 400 台 scintillator array + muon detector に対し、primary cosmic ray energy を mass-independent に再構成する GNN pipeline。従来の electron size calibration curve より高精度化を狙い、feature mapping を自動化する modular/dynamic GNN を実装。latent space の解釈とスケーリングによる response resolution 改善も議論。air-shower 系だが、sparse detector array の event representation と reconstruction bias 評価の話として参考になる。
- **キーワード**: cosmic ray, energy reconstruction, GNN, GRAPES-3, detector array

### [15] Observation of the doubly charmed baryon $\Xi_{cc}^+$ with the LHCb Run 3 detector
- **著者**: LHCb Collaboration
- **リンク**: https://arxiv.org/abs/2603.28456
- **要約**: LHCb Run 3、13.6 TeV pp、2024 年データ 6.9 fb$^{-1}$ で $\Xi_{cc}^+\to\Lambda_c^+K^-\pi^+$ を 7σ 超で初観測。質量は $3619.97 \pm 0.83 \pm 0.26 ^{+1.90}_{-1.30}$ MeV/$c^2$。$\Xi_{cc}^{++}$ との差も測定。LHCb Run 3 での初の新粒子観測という意味で実験的インパクトが大きい。ニュートリノではないが、Run 3 trigger/reconstruction の実戦投入が物理成果につながった例。
- **キーワード**: LHCb, Run 3, doubly charmed baryon, $\Xi_{cc}^+$, heavy flavor

### [16] Source of Internal Crosstalk in Silicon Photomultipliers
- **著者**: L. Wang, H. Lewis, F. Retière, A. de St Croix
- **リンク**: https://arxiv.org/abs/2509.05450
- **要約**: SiPM の internal crosstalk 起源を、温度・overvoltage 依存性と charge-carrier type ごとの avalanche triggering probability から診断。crosstalk photon が n-doped region で吸収されると hole-triggered avalanche、p-doped region なら electron-triggered avalanche になる点を使い、device structure と crosstalk source を結びつける。表面反射が bulk crosstalk mitigation 後の主要成分になり得ると指摘。次世代 dark matter / neutrino detector の photon sensor noise model に直結する実用的な論文。
- **キーワード**: SiPM, internal crosstalk, photon sensor, neutrino detector, dark matter detector

### [18] High-Energy Neutrino Tomography of the Earth's Interior with IceCube
- **著者**: IceCube Collaboration
- **リンク**: https://arxiv.org/abs/2607.02644
- **要約**: IceCube の 10.7 年分の主に muon-neutrino data、500 GeV–100 TeV を使い、地球通過時の neutrino attenuation から地球内部の radial density profile を推定。大気ニュートリノと diffuse astrophysical neutrino flux を含め、zenith・energy 依存の flux suppression を、flux model・cross section・detector response・氷の systematic を含む shell model fit で解析。PREM や重力測定と整合し、弱相互作用による地球質量・慣性モーメント測定としては最精密。ニュートリノを「地球物理プローブ」として使う方向の非常にきれいな実証。
- **キーワード**: IceCube, neutrino tomography, Earth density, atmospheric neutrino, attenuation

### [19] Trigger system for the Payload for Ultrahigh Energy Observations (PUEO) balloon-borne neutrino detector
- **著者**: Q. Abarr et al.
- **リンク**: https://arxiv.org/abs/2607.05285
- **要約**: UHE neutrino balloon experiment PUEO の trigger system 論文。Askaryan emission または upward-going air shower の radio signal を狙い、24 個の Xilinx RFSoC digitizer、192 channels、3 GSa/s sampling、375 MHz system clock の fully digital supersample-rate beamformer を構成。frequency-band conditioning、動的 RFI rejection、matched filtering を実装し、48 synthetic beams で $\sim25^\circ$ zenith × $\sim60^\circ$ azimuth をカバー。preflight test では simulated signal に対して最小 SNR $\sim1.5$、trigger logic 消費電力 5–7 W。UHE ν の radio trigger として、低電力 FPGA/RFSoC 実装の具体性が高い。
- **キーワード**: PUEO, UHE neutrino, RFSoC, beamforming trigger, Askaryan, RFI rejection
