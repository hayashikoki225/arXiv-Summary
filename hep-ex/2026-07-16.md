### [1] A Low-energy Threshold and Multi-messenger Trigger System for the JUNO Experiment
- **著者**: JUNO Collaboration
- **リンク**: https://arxiv.org/abs/2607.13427
- **要約**: JUNO の標準 global trigger を補完する、低閾値・multi-messenger trigger system の実装報告。2025年運転で有効エネルギー閾値 $110 \pm 10$ keV を達成し、将来的には 100 keV を十分下回る可能性を示す。外部 alert、特に重力波観測などを受けて transient neutrino burst をリアルタイム監視する仕組みも統合されており、core-collapse supernova burst に対して約 250 kpc まで感度を持つ。JUNO を単なる reactor $\bar\nu_e$ 精密測定装置から、MeV transient neutrino observatory に拡張するための実験的インフラ論文。低閾値化が solar/geo/supernova/BSM low-energy channel に効くので、ニュートリノ実験としてかなり重要。
- **キーワード**: JUNO, low-energy trigger, multi-messenger, supernova neutrino, real-time monitor

### [2] Measurement of $\Xi^-/\bar\Xi^{+}$ production in jets from $Z$ boson decays with the DELPHI open data
- **著者**: Zhouyue Fan, Yuzhi Che, Xinchou Lou, Zhaoru Zhang, Jinfei Wu, Gengyuan Zhang, Gang Li, Yanping Huang, Manqi Ruan
- **リンク**: https://arxiv.org/abs/2607.13804
- **要約**: LEP/DELPHI open data の 320 万 $Z\to$ hadrons 事象を用い、$Z$ 崩壊 jet 中の double-strangeness baryon $\Xi^-/\bar\Xi^+$ 生成を測定。Durham algorithm, $y_{\rm cut}=0.005$ で jet を再構成し、3-jet event の energy ordering から quark/gluon-enriched sample を作る。gluon jet と quark jet の $\Xi$ 生成比を荷電粒子 multiplicity で規格化した量として $R^{\Xi^-}_{g/q}=1.21\pm0.18{\rm(stat.)}\pm0.26{\rm(syst.)}$ を得て、JETSET や OPAL の $K_S^0,\Lambda$ 結果と整合。open data 再解析としては堅実で、non-perturbative QCD fragmentation、特に baryon/strangeness 生成の validation に使える。
- **キーワード**: DELPHI open data, $\Xi$ baryon, jet fragmentation, strangeness, LEP

## Cross-list

### [3] The soft volume of ultra-high energy neutrinos experiments
- **著者**: Stefano Palmisano, Diego Redigolo, Michele Tammaro, Andrea Tesi
- **リンク**: https://arxiv.org/abs/2607.13143
- **要約**: UHE neutrino telescope の through-going muon event rate を、検出器体積だけでなく muon propagation が作る有効な「soft volume」として定式化する半解析的フレームワーク。secondary muon の Boltzmann equation から出発し、soft energy loss が支配的であることを使って drift-diffusion 型に落とし、rare hard scattering は摂動的に扱う。IceCube through-going muon data に適用し、KM3NeT の ultra-high-energy track event と IceCube 非観測の整合性も再検討。フル MC に頼らず flux model と track observable をつなぐ道具として有用で、UHE neutrino astronomy や BSM search の rate reinterpretation に効きそう。
- **キーワード**: UHE neutrino, IceCube, KM3NeT, through-going muon, effective volume

### [4] Weak charged current induced electron and positron scattering off proton at JLab and MAMI energies
- **著者**: A. Fatima, M. Sajjad Athar, S. K. Singh
- **リンク**: https://arxiv.org/abs/2607.13523
- **要約**: JLab/MAMI エネルギー領域での $e^\pm p$ weak charged-current scattering を包括的に扱う review。数百 MeV から数 GeV の中間エネルギーで、quasielastic、strangeness-changing channel、$P_{33}(1232)$, $P_{11}(1440)$, $S_{11}(1535)$ resonance、$\eta/K$ meson、strange particle associated production まで整理する。ニュートリノ散乱で問題になる axial form factor や axial dipole mass を、charged lepton beam で独立に制約するという文脈が重要。直接のニュートリノ実験論文ではないが、neutrino-nucleus interaction model の外部制約として読む価値がある。
- **キーワード**: weak charged current, JLab, MAMI, axial form factor, neutrino interaction model

### [5] Polarized observables in $\tau\to VV\nu_\tau$ decays
- **著者**: Vindhyawasini Prasad, Hiroyuki Umeeda
- **リンク**: https://arxiv.org/abs/2607.13743
- **要約**: $\tau^- \to V_1^- V_2^0 \nu_\tau$ 崩壊の polarization observables を helicity formalism と generalized hidden local symmetry framework で解析。対象は $\rho\rho$, $\rho\omega$, $K^*\rho$, $K^*\omega$, $\rho\bar K^*$ など。longitudinal polarization fraction $f_L$ が endpoint limit の $1/3$ から有意にずれることを示し、Belle II 測定で charm nonleptonic decay に見える polarization pattern との比較が可能になる。hep-ex 的には Belle II phenomenology 寄り。
- **キーワード**: tau decay, polarization, Belle II, helicity formalism

### [6] RI/SMOM renormalization for lattice QCD: bilinear and three-quark operators
- **著者**: Bernd A. Kniehl, Oleg L. Veretin
- **リンク**: https://arxiv.org/abs/2607.13750
- **要約**: lattice QCD の RI/SMOM と $\overline{\rm MS}$ scheme 間 matching の review/proceedings。bilinear operator では local quark current や structure function moment の 3-loop conversion factor、three-quark operator では baryon distribution amplitude に関係する 2-loop/3-loop 結果を整理。hep-ex への直接性は薄いが、将来の hadron structure や baryon distribution amplitude の理論入力精度に関係する。
- **キーワード**: lattice QCD, RI/SMOM, renormalization, baryon distribution amplitude

### [7] Evidence for sequential $\Upsilon$(nS) suppression in light ion collisions
- **著者**: CMS Collaboration
- **リンク**: https://arxiv.org/abs/2607.13758
- **要約**: CMS による O-O および Ne-Ne 衝突での $\Upsilon$(1S,2S,3S) 測定。重イオン衝突で QGP の典型的シグナルとされる quarkonium sequential suppression が、より軽い ion collision でも見えるかを調べる。$\Upsilon$(2S)/$\Upsilon$(1S) は pp reference より有意に低く、$\Upsilon$(3S)/$\Upsilon$(1S) はさらに大きく低下、$\Upsilon$(3S) の $\Upsilon$(2S) に対する相対 suppression は 3σ 超。小さい系での QGP-like medium formation を議論する上で重要な結果。
- **キーワード**: CMS, light ion collision, bottomonium, sequential suppression, QGP

### [8] Mono-Z Dark Matter Search with Neural Spline Flows Using CMS Run 2015D Open Data
- **著者**: Hitesh Rasineni, Bhavishya Chebrolu
- **リンク**: https://arxiv.org/abs/2607.13771
- **要約**: CMS Run 2015D open data 2.32 fb$^{-1}$ を使った mono-$Z\to\ell^+\ell^-$ dark matter search。37 次元の kinematic feature に対し Neural Spline Flow を用いて signal/background density を学習し、log-likelihood ratio を event-level statistic として構成する。hard MET cut に強く依存せず位相空間全体を使う発想が新規性。結果として scalar/vector/axial-vector mediator で 95% CL 上限を出すが、observed limit は high-MET background modeling discrepancy のため expected より弱い。open data + likelihood-ratio ML の実証として面白いが、物理結論は慎重に見るべき。
- **キーワード**: CMS open data, mono-Z, dark matter, Neural Spline Flow, likelihood ratio

### [9] Radiation effects on surface and bulk properties of ATLAS18 silicon sensors under low- and high-dose gamma irradiation and annealing
- **著者**: Marcela Mikestikova et al.
- **リンク**: https://arxiv.org/abs/2607.13932
- **要約**: HL-LHC ATLAS ITk 用 ATLAS18 silicon strip sensor の gamma irradiation と annealing の系統的評価。low TID 0.5–100 krad は ITk 初期運転に対応し、さらに数 Mrad から数百 Mrad まで拡張して surface/bulk damage を分離して調べる。leakage current、CV からの full depletion voltage、80–300°C の isochronal annealing、60/160°C の isothermal annealing を測定。検出器運用・品質管理・radiation damage modeling の実務的価値が高い。
- **キーワード**: ATLAS ITk, silicon strip sensor, gamma irradiation, annealing, HL-LHC

### [10] Prospects for Exploring Non-Standard Neutrino Properties with Argon-Based CEvNS Experiments
- **著者**: Sam Carey, Vishvas Pandey
- **リンク**: https://arxiv.org/abs/2510.14015
- **要約**: stopped-pion source に置かれた argon-based CEvNS detector で、SM precision test と non-standard neutrino property をどこまで探れるかを評価。CENNS-10, CENNS-750, CCM, PIP2-BD/F2D2 を具体的に扱い、現実的な flux と detector configuration から event rate と感度を見積もる。対象は weak mixing angle の低 $Q^2$ 測定、neutrino magnetic moment、effective charge radius、quark への NSI。argon CEvNS は clean で拡張性のある低エネルギー electroweak/BSM probe だ、という立場が明確。DUNE 系 LAr 技術や stopped-pion program とも接続するので、ニュートリノ BSM phenomenology と実験設計の橋渡しとして有用。
- **キーワード**: CEvNS, argon detector, stopped-pion source, NSI, neutrino electromagnetic properties
