### [1] Track fitting at the full LHC collision rate
- **著者**: Pierre Billoir, Thomas Boettcher, Michel De Cian, Lennart H. Uecker
- **リンク**: https://arxiv.org/abs/2607.14793
- **要約**: LHCb Run 3 の全ソフトウェアトリガー第1段で使う、30 MHz フル衝突頻度対応の高速 Kalman track fitter。磁場マップ lookup と数値積分を高速な解析的パラメータ化に置き換え、約500 GPU 上で実運用。処理時間増は従来比 +2% に抑えつつ、$D^0$ と $J/\psi$ の不変質量分解能を約2倍改善し、misalignment にもより robust。リアルタイム再構成で「物理性能を落とさず高速化」ではなく、むしろ resolution と fake rejection を上げている点が強い。
- **キーワード**: LHCb, real-time reconstruction, Kalman filter, GPU trigger, tracking

### [2] A Differential Neutral-to-Charged Current Double-Proton Observable for Studying Short-Range Correlations in Neutrino--Argon Scattering
- **著者**: A. Bueno, D. Garcia-Gamez, C. Martin-Morales
- **リンク**: https://arxiv.org/abs/2607.14942
- **要約**: neutrino–argon での 2 nucleon emission、特に SRC 成分を直接制限するため、NC/CC の 2 proton production 比を proton-pair relative momentum の関数として測る提案。CC では豊富な np-SRC が weak interaction により pp-like visible pair を作れる一方、NC では pp-SRC が少ないため同様の寄与が抑制される、という isospin 構造を使う。flux normalization や相関 detector systematic が比でかなり cancel するので、既存/近未来の LArTPC で実現可能という主張。DUNE 系の neutrino interaction modeling に直結する、かなり読む価値のある observable 提案。
- **キーワード**: neutrino-argon, LArTPC, SRC, 2p emission, NC/CC ratio

### [3] The study of bulk damage of high gamma-irradiated $n^+$-in-$p$ silicon diodes
- **著者**: M. Mikestikova et al.
- **リンク**: https://arxiv.org/abs/2607.15056
- **要約**: $^{60}$Co gamma で 0.50–8.28 MGy まで照射した $n^+$-in-$p$ silicon diode の bulk damage を IV/CV と Transient Current Technique で評価。leakage current は total ionizing dose にほぼ線形に増え、damage coefficient は初期 resistivity と oxygen concentration に依存。effective doping concentration と full depletion voltage は一度大きく下がってから上がる振る舞いを示し、acceptor removal が示唆される。HL-LHC など高放射線環境の silicon sensor 寿命評価として堅実。
- **キーワード**: silicon sensor, gamma irradiation, bulk damage, TID, acceptor removal

### [6] Searching for the $N$-naturalness tower of neutrinos
- **著者**: Sofia Lonardi, Manuel Ettengruber, Philipp Eller
- **リンク**: https://arxiv.org/abs/2607.14243
- **要約**: $N$-naturalness model に現れる多数の dark-sector neutrino tower を、公開 neutrino data の global analysis で初探索。normal/inverted ordering、Majorana/Dirac の場合に $(r,N)$ 平面で制限を出し、Majorana では $r\geq0.1$ に対して $N\leq10^4$ を排除可能とする。これは GUT scale 付近での gauge/gravity unification を動機にした benchmark を直接潰す、という意味で phenomenology 色は強いが neutrino data の使い方として面白い。
- **キーワード**: neutrino, N-naturalness, dark sectors, global fit, hierarchy problem

### [7] Comprehensive study of timing resolution in plastic scintillator detectors with wavelength-shifting fiber and silicon photomultiplier readout
- **著者**: Haohui Che, Guang Yang
- **リンク**: https://arxiv.org/abs/2607.14290
- **要約**: plastic scintillator + WLS fiber + SiPM readout の timing resolution を、semi-analytical model、toy MC、Geant4 optical simulation で包括的に調べた研究。scintillation rise/decay、WLS re-emission、fiber transport、SiPM SPTR、electronics quantization まで chain 全体を分解して timing map を作る。7 scintillator、8 WLS fiber、5 SiPM、5 electronics、3 readout topology の scan をしており、detector design の lookup table として有用。ニュートリノ検出器の veto / timing layer / near detector 系にも応用しやすい。
- **キーワード**: plastic scintillator, WLS fiber, SiPM, timing resolution, Geant4

### [10] A flexible DAQ system for the Timepix4 ASIC in the 4DPHOTON project
- **著者**: N. V. Biesuz et al.

### [12] Comparative study and optimization of SDHCAL hadronic energy reconstruction methods
- **著者**: Tanguy Pasquier, Gérald Grenier, Imad Laktineh
- **リンク**: https://arxiv.org/abs/2607.15023
- **要約**: ILD detector concept の Semi-Digital Hadronic Calorimeter について、single $K^0_L$ と dijet event で hadronic energy reconstruction を比較。linear、quadratic、split、polynomial regression 型の再構成式を検討し、split method と polynomial regression が低エネルギー resolution と高エネルギー linearity のバランスで良い。高エネルギー dijet では PFA confusion が支配的で、将来は timing 情報を APRIL に統合することで改善余地がある。次世代 lepton collider calorimetry の実務的 study。
- **キーワード**: SDHCAL, ILD, calorimetry, PFA, hadronic energy reconstruction

### [13] Radiative corrections in neutral-current (anti)neutrino elastic scattering at $\text{GeV}$ energies I: Nucleon targets
- **著者**: Yi Chen, Oleksandr Tomalak, Bing-Song Zou
- **リンク**: https://arxiv.org/abs/2607.15050
- **要約**: GeV 領域の NC (anti)neutrino–nucleon elastic scattering に対する radiative correction を EFT で整理。soft/hard function に factorize し、振幅・断面積レベルでの neutrino flavor dependence と low-energy NC に入る QCD contribution を明確化。single nucleon level の補正は strange-quark contribution と同程度になり得るため、NC elastic から axial strange や BSM を読む解析では無視しにくい。BNL E734 / MiniBooNE data とも良く合う。neutrino scattering systematics の観点でかなり重要。
- **キーワード**: neutral-current neutrino scattering, radiative correction, EFT, GeV neutrino, MiniBooNE

### [14] Next-to-next-to-leading order QCD corrections to pion (kaon)-induced exclusive Drell-Yan process
- **著者**: Yu Jia, Bernard Pire, Qin-Tao Song, Guang Tang, Zhe-Yu Wang
- **リンク**: https://arxiv.org/abs/2607.15214
- **要約**: J-PARC など将来の high-energy pion/kaon beam 実験を念頭に、$\pi^-p\to\gamma^*(\to\ell^+\ell^-)+n$ と $K^-p\to\gamma^*(\to\ell^+\ell^-)+\Lambda$ の exclusive Drell-Yan に対する NNLO QCD correction を計算。GPD factorization の leading twist で扱い、NNLO 補正は substantial かつ positive と結論。実験側から見ると、将来データと比較する理論予言には NNLO が必須、というメッセージ。
- **キーワード**: J-PARC, pion beam, kaon beam, exclusive Drell-Yan, NNLO QCD

### [17] Probing unexplored spin-dependent dark matter-proton coupling with few-photoelectron threshold in COSINE-100
- **著者**: COSINE-100 Collaboration
- **リンク**: https://arxiv.org/abs/2603.22411
- **要約**: COSINE-100 の NaI(Tl) データで、低質量 DM–proton spin-dependent scattering を few-photoelectron threshold まで攻める解析。MLP による event selection と noise mitigation により、従来 8 photoelectron threshold だった領域を 3/4 isolated peak まで下げる。annual modulation は見えず、1.75–2.25 GeV/$c^2$ で世界最強の SD DM-proton 制限、Migdal effect を含めると 15–58 MeV/$c^2$ でも world-leading と主張。低閾値化の実験技術と background modeling が肝。
- **キーワード**: COSINE-100, low-mass dark matter, spin-dependent, NaI(Tl), Migdal effect
