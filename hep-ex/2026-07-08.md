### [1] Gamma Backgrounds for Experiments at the High Flux Isotope Reactor
- **著者**: M. Andriamirado et al.
- **リンク**: https://arxiv.org/abs/2607.05834
- **要約**: ORNL の High Flux Isotope Reactor (HFIR) 実験ホールで、Ge 検出器を用いてガンマ線背景のエネルギー分布・空間分布・時間変動を系統的に測定。PROSPECT がデータ取得した場所、および将来の炉ニュートリノ実験候補地点を念頭に、炉起源・中性子ビームライン起源・建材起源の寄与を分離している。数十 keV 領域の背景低減に対する遮蔽評価も含み、IBD ベースの炉ニュートリノ測定や CEvNS 実験の site characterization としてかなり実用的。ニュートリノ屋目線では、低エネルギー ER/NR 背景の現場情報が具体的にまとまっている点が価値。
- **キーワード**: HFIR, reactor neutrino, gamma background, PROSPECT, CEvNS, shielding

### [2] A unified quenching model in Geant 4 simulation for $\alpha$, proton and electron particles in liquid scintillator detectors
- **著者**: Shubing Liu, Mingyuan Wang, Zeyuan Yu
- **リンク**: https://arxiv.org/abs/2607.05875
- **要約**: 液体シンチレータ検出器の quenching を、粒子種ごとに異なる Birks 係数で合わせる従来の扱いに対し、Geant4 内で二次電子、特に 4 keV 以下の $\delta$ electron をより明示的に追跡すれば、$\alpha$/proton/electron を単一の Birks 係数 $k_B \simeq 0.013~{\rm g\,cm^{-2}\,MeV^{-1}}$ で記述できると主張。JUNO/SNO+ のような精密エネルギースペクトル実験では、粒子種依存の経験的チューニングを減らし、bench-top 測定間の比較を揃える可能性がある。これはニュートリノ振動解析のエネルギー応答系統に直結するので要チェック。
- **キーワード**: liquid scintillator, quenching, Birks law, Geant4, JUNO, SNO+

### [3] Observation and branching fraction measurements of $J/\psi\to p \bar p K^0_S K^0_S$ and $\psi(3686) \to p \bar p K^0_S K^0_S$
- **著者**: BESIII Collaboration
- **リンク**: https://arxiv.org/abs/2607.06070
- **要約**: BESIII の巨大統計、$10.087\times10^9$ $J/\psi$ と $2.712\times10^9$ $\psi(3686)$ 事象を用い、$J/\psi,\psi(3686)\to p\bar p K^0_S K^0_S$ をともに $10\sigma$ 超で初観測。分岐比はそれぞれ $(1.60 \pm 0.02 \pm 0.09)\times10^{-5}$、$(3.93 \pm 0.24 \pm 0.34)\times10^{-6}$。比は約 24.6% で、単純な理論期待から 4.6σ ずれる。ニュートリノとは遠いが、チャーモニウム崩壊・baryon pair 生成・12% rule まわりの精密データとしては堅い。
- **キーワード**: BESIII, charmonium, branching fraction, baryon pair, $K^0_S$

### [4] Simultaneous efficiency measurements of $b$- and $c$-jets in $t\bar{t}$ events from $\sqrt{s}=13.6$ TeV $pp$ collision data collected with the ATLAS detector
- **著者**: ATLAS Collaboration
- **リンク**: https://arxiv.org/abs/2607.06199
- **要約**: ATLAS Run 3 の 56 fb$^{-1}$ を使い、現在使われている single-transformer ベースの GN2 tagger について、$b$ tagging efficiency と $c\to b$ mistag efficiency を $t\bar t$ 事象中で同時測定。kinematic likelihood で jet と $t\bar t$ 崩壊生成物を対応付け、leptonic top の $b$ jet と hadronic $W\to cs$ 由来 jet を利用する。精度は $b$ jet で 2% 未満、$c$ mistag で 5% 程度。新物理探索そのものではないが、Transformer tagger の実データ calibration として重要。
- **キーワード**: ATLAS, GN2, transformer, b-tagging, c-mistag, Run 3

### [5] Efficiency measurements of GEM GE1/1 chambers in the upgraded CMS Endcap Muon System using 2023 collision data at $\sqrt{s}=13.6$ TeV
- **著者**: CMS GEM Collaboration
- **リンク**: https://arxiv.org/abs/2607.06353
- **要約**: CMS forward muon upgrade の GE1/1 triple-GEM chamber について、2023年 13.6 TeV pp collision data、17.8 fb$^{-1}$ の $Z\to\mu\mu$ enriched sample で検出効率を測定。GEM を使わずに他の tracking/muon detector で muon trajectory を外挿し、137台の chamber の平均効率を評価。全体平均は約 93.3%、short のない 108台を nominal HV で見ると約 96%。pile-up に効率が依存しないという結果は、HL-LHC forward muon 環境に向けた実証として素直に重要。
- **キーワード**: CMS, GEM, muon detector, GE1/1, HL-LHC, detector efficiency

## 注目 cross-list

### [6] Searching for charged Higgs bosons in top decays via the $t^*b$ channel
- **著者**: Saiyad Ashanujjaman, Guglielmo Coloretti, Andreas Crivellin, Benjamin Fuks, Yeonsu Heo
- **リンク**: https://arxiv.org/abs/2607.05499
- **要約**: 軽い charged Higgs について、従来の $H^\pm\to\tau\nu,cs,cb$ ではなく、top threshold 以下で重要になり得る $H^\pm\to t^*b$ に注目。$t\to H^\pm b$、続く $H^\pm\to t^*b$ により $t\bar t b\bar b$ 的な終状態が生じるため、ATLAS の dileptonic fiducial 測定を再解釈して制限を置く。$m_{H^\pm}=110$–165 GeV で ${\rm Br}(t\to H^\pm b)\times{\rm Br}(H^\pm\to t^*b)$ に 1.9–2.9% の上限。専用解析を促す phenomenology paper として明快。
- **キーワード**: charged Higgs, top decay, $t^*b$, 2HDM, ATLAS reinterpretation

### [7] Characterization of p-stop isolation implants in silicon sensors using MOSFET structures
- **著者**: Thomas Bergauer et al.
- **リンク**: https://arxiv.org/abs/2607.05528
- **要約**: p-type silicon sensor の n-type electrode 間を分離する p-stop implant を、MOSFET test structure で非破壊的に評価する手法。backside voltage 依存の transfer characteristic から threshold voltage を取り出し、そこから depth-dependent doping profile や inter-electrode isolation を推定する。HL-LHC や将来 collider detector の silicon sensor QA/QC として地味だが実用性が高い。検出器開発寄りで、量産時の wafer 間ばらつき監視に効きそう。
- **キーワード**: silicon sensor, p-stop, MOSFET, isolation implant, detector QA

### [8] $K^{*}(892)^0$ production and the time between freeze-outs in $^{40}$Ar+$^{45}$Sc collisions by NA61/SHINE at the CERN SPS
- **著者**: NA61/SHINE Collaboration
- **リンク**: https://arxiv.org/abs/2607.05595
- **要約**: SPS の $^{40}$Ar+$^{45}$Sc central collision で $K^{*}(892)^0$ 生成を初測定。$\sqrt{s_{\rm NN}}=8.8, 11.9, 16.8$ GeV のデータを使い、$\langle K^{*0}\rangle/\langle K^\pm\rangle$ 比から chemical freeze-out と kinetic freeze-out の時間差を推定する。重イオン・QCD 相図寄りなのでニュートリノからは遠いが、短寿命共鳴を「ハドロン相の時計」として使う標準的な考え方の実データ更新。
- **キーワード**: NA61/SHINE, heavy ion, $K^{*0}$, freeze-out, SPS

### [9] Articulating Assumptions in AI-Generated Scientific Analyses through Task Decomposition
- **著者**: Ahmed Hammad, Mihoko Nojiri
- **リンク**: https://arxiv.org/abs/2607.05762
- **要約**: LLM が生成した科学解析コードについて、「何を計算しているか」「どの仮定が結果を決めているか」を明示化する multi-agent framework。コード生成・実行・tracing・validation を分け、量に紐づいた semantic differencing で intended analysis と implemented code の差を検出する。collider physics 解析で検証されているが、ニュートリノ解析にもそのまま刺さる問題設定。特に selection, weighting, nuisance handling などが自然言語仕様から曖昧に実装されるリスクを扱う点が現代的。
- **キーワード**: LLM, scientific analysis, reproducibility, semantic differencing, collider analysis

### [10] ggxy: Fast and flexible NLO QCD corrections to gluon-initiated processes
- **著者**: Daniel Stremmer
- **リンク**: https://arxiv.org/abs/2607.06099
- **要約**: $gg\to HH$、新たに導出した $gg\to ZH$、$gg\to ZZ$ など、top loop を含む gluon-initiated process の NLO QCD 断面積を高速に計算する C++ ライブラリ `ggxy` の紹介。two-loop virtual amplitude は運動学領域ごとの解析近似を使い、それ以外は exact に扱う。Powheg interface により parton shower matching も可能。LHC の Higgs pair / diboson / EFT 解析のバックエンドとして有用そう。
- **キーワード**: NLO QCD, gluon fusion, $gg\to HH$, Powheg, collider tools
